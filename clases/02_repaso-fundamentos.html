Algoritmo

  • Un algoritmo es una secuencia finita, ordenada y definida de pasos que permiten resolver un problema específico o alcanzar un objetivo determinado.
  • Debe cumplir con las siguientes propiedades:
    • Finitud: debe tener un número determinado de pasos.
    • Precisión: cada instrucción debe ser clara y no ambigua.
    • Eficiencia: debe permitir alcanzar una solución de manera razonable en tiempo y recursos.
  • Ejemplo: Una receta muy detallada de cocina

Pseudocódigo

  • El pseudocódigo es una representación textual y estructurada de un algoritmo, redactada con un lenguaje intermedio entre el lenguaje natural y un lenguaje de programación formal.
  • Su propósito es facilitar la construcción lógica de soluciones antes de codificarlas, sin necesidad de una sintaxis estricta.

Ejemplo: El pseudocódigo para decir si un número es positivo

Inicio
  Leer número
  Si número > 0 entonces
    Escribir "Positivo"
  Sino si número < 0 entonces
    Escribir "Negativo"
  Sino
    Escribir "Cero"
Fin

Código fuente

  • El código fuente es la implementación formal de un algoritmo utilizando un lenguaje de programación.
  • Dicho código debe respetar la sintaxis del lenguaje utilizado y puede ser interpretado o compilado por una máquina para su ejecución.

Ejemplo: Código en Python para decir si un número es positivo

numero = int(input("Ingrese un número: "))
if numero > 0:
    print("Positivo")
elif numero < 0:
    print("Negativo")
else:
    print("Cero")

Python

  • Python es un lenguaje de programación de alto nivel, interpretado, multiparadigma (soporta programación estructurada, orientada a objetos y funcional), diseñado para ser legible, expresivo y extensible.

  • Fue desarrollado en la década de 1990 por Guido Van Rossum, con el objetivo de ser fácil de aprender para principiantes y aún así poderoso para usuarios avanzados.

  • El nombre de Python proviene del programa de televisión favorito de Guido, “Monty Python’s Flying Circus”.

Ventajas de Python

  • Sintaxis clara y concisa.
  • Ampliamente utilizado en la industria y academia.
  • Amplia comunidad y ecosistema de bibliotecas.
  • Portabilidad y versatilidad en múltiples dominios: desarrollo web, inteligencia artificial, análisis de datos, automatización, etc.

Editor vs. IDE

Editor de código

Un editor de código es una herramienta de software utilizada para escribir y editar código fuente.

Funcionalidades comunes:

  • Coloreado de sintaxis
  • Autocompletado
  • Identación automática

Ejemplos: VSCode, Sublime Text, Notepad++

Entorno de desarrollo (IDE)

Un IDE es una plataforma que unifica múltiples herramientas necesarias para el desarrollo.

Componentes comunes:

  • Editor de código y consola
  • Depurador (debugger)
  • Explorador de archivos
  • Visualizador de variables

Ejemplos: Spyder, PyCharm, Thonny

Spyder será el IDE utilizado en este curso. Está orientado a desarrollo científico y educativo, y forma parte de la distribución Anaconda.

Anaconda

Anaconda es una distribución de Python ampliamente utilizada que incluye:

  • Un conjunto de bibliotecas preinstaladas.
  • Entornos gráficos para desarrollo (Spyder, Jupyter).
  • conda, su propio gestor de paquetes.
  • Herramientas para crear y gestionar entornos virtuales.

¿Qué es un gestor de paquetes?

Es una herramienta que permite instalar, actualizar y administrar bibliotecas y dependencias externas al lenguaje.

Nota: Mientras Anaconda usa conda, el gestor estándar de Python es pip.

Ambiente virtual

Un ambiente virtual es una instancia aislada del intérprete de Python y sus paquetes, utilizada para gestionar proyectos de manera independiente.

¿Por qué utilizarlos?

  • Evita conflictos: Impide que las actualizaciones de un proyecto rompan otro que requiere versiones anteriores.
  • Reproducibilidad: Permite que otros colaboradores instalen exactamente las mismas versiones de paquetes.
  • Limpieza: Facilita la instalación controlada de dependencias sin “ensuciar” la instalación global de Python en el sistema.

Estructura de un programa

  1. Comentario: Texto que no afecta la ejecución. Sirve para aclaraciones.
# Una sola linea
""" Varias 
    lineas """
  1. Sentencia: Instrucción que realiza una acción. El salto de línea (Enter) indica el fin.
x = 10
  1. Indentación: Sangría obligatoria para definir bloques de código en lugar de llaves {}.
if x > 0:
    print("Identado")
  1. Función y llamada: Bloque reutilizable (def) y su invocación por nombre.
def suma(a, b): 
    return a + b
res = suma(3, 5)
  1. Expresión: Combinación de valores y operadores que producen un resultado (ej. 3 + 5).
  2. Bloque: Conjunto de sentencias agrupadas por la misma indentación.

Variables y tipos de datos

Variable

Una variable es un nombre asociado a un valor almacenado en memoria.

  • Python permite declarar variables sin indicar su tipo explícitamente (tipado dinámico).
  • Los nombres deben ser descriptivos y no pueden iniciar con números ni contener espacios.
nombre = "Ana"       # Cadena de texto (str)
edad = 25            # Entero (int)
altura = 1.68        # Decimal (float)
es_estudiante = True # Booleano (bool)

Tipos de datos básicos

Clasificación

Numéricos

  • 🔴 int: números enteros.
  • 🔵 float: punto flotante.
  • 🟣 bool: lógicos (True/False).

Mapeo

  • 🔼 dict: pares clave-valor.

Secuencias

  • 🟨 list: mutable y ordenada.
  • 🟧 tuple: inmutable y ordenada.
  • 🟫 str: texto inmutable.

Especiales

  • 🔻 None: ausencia de valor.

🔴 Enteros (int)

Los enteros son un tipo de dato que permite representar números enteros, positivos o negativos, sin decimales.

Sistemas de numeración:

  • Binario: Prefijo 0b (ej., 0b101).
  • Hexadecimal: Prefijo 0x (ej., 0xA1).
  • Octal: Prefijo 0o (ej., 0o17).
  • Al imprimir, el número se convierte a decimal.
num_decimal = 42
num_binario = 0b101
num_hexadecimal = 0xA1
num_octal = 0o17

print(num_decimal, num_binario, num_hexadecimal, num_octal)
# Esto imprime: 42 5 161 15

🔴 Enteros - Recordatorio

📌 Recordatorio:

  • Python asigna bits según el valor del número. Puedes usar sys.getsizeof() para conocer el tamaño en memoria.
  • Puedes convertir otros tipos a int. Los decimales se truncan al convertir a int.
# ¿Cuál es la salida de este código?
import sys

num = 4
print(sys.getsizeof(num))

num_decimal = 3.7
num_entero = int(num_decimal)
print(num_entero)

🔵 Flotantes (float)

Representan números positivos o negativos con decimales.

No tienen precisión infinita. Los valores máximos y mínimos están definidos por el sistema, aunque podemos hacer una representación simbólica del número infinito y NaN.

# Float
num1 = 3.14159
print(num1, type(num1))

# Infinito y NaN
num_infinito = float('inf')
num_nan = float('nan')
print(num_infinito, num_nan)

# Mínimo y máximo
import sys
print("Valor mínimo", sys.float_info.min)
print("Valor máximo", sys.float_info.max)

# ¿Cuál es el resultado de esta operación?
num2 = 1-2.2250738585072014e-308
print(num2)

🔵 Flotantes - Recordatorio

📌 Recordatorio:

  • Podemos indicar números flotantes mediante notación científica, usamos e para indicar el exponente.
  • Podemos convertir otros tipos a float usando float().
num3 = 1.2e3  # Notación científica (1200.0)
print(num3)

🟣 Booleanos (bool)

  • Tipo de dato que almacena True o False.
  • Se utiliza con operadores como >, <, ==, que devuelven valores booleanos.
  • Podemos convertir valores a bool usando bool().
valor1 = True
valor2 = (5 > 3)  # True
valor3 = bool(0)   # False
valor4 = bool(1)   # True
print(valor1, valor2, valor3, valor4)

🟨 Listas (list)

Estructura de datos mutable y dinámica.

Propiedades:

  • Ordenadas.
  • Contienen tipos arbitrarios.
  • Se crean e indexan con [].
  • Pueden anidarse.
mi_lista = [1, 2, 3, "Python", [5, 6]]
print(mi_lista[0])      # Salida?
print(mi_lista[-1])     # Salida?
mi_lista[1] = "Nuevo"
print(mi_lista)         # Salida?

🟨 Listas - Métodos comunes

📌 Recordatorio:

Métodos comunes de las listas:

  • append(valor): Agrega elemento al final.
  • extend(iterable): Agrega elementos de un iterable.
  • insert(pos, valor): Inserta en una posición.
  • pop(indice): Elimina y devuelve el elemento en el índice (por defecto el último).
  • remove(valor): Elimina la primera aparición del valor.
  • reverse(): Invierte el orden.
  • sort(): Ordena la lista.
mi_lista2 = [3, 1, 4]
mi_lista2.append(5)
mi_lista2.sort()
print(mi_lista2)  # Salida?

🟧 Tuplas (tuple)

Estructura de datos inmutable. Una vez que una tupla ha sido creada, sus elementos no pueden ser modificados.

Propiedades:

  • Inmutables.
  • Ordenadas.
  • Contienen tipos arbitrarios.
  • Se crean con (), se indexan con [].
  • Pueden anidarse.
  • Pueden contener duplicados.
mi_tupla = (10, 20, 30, 40)
print(mi_tupla[0])  # Imprime: 10
print(mi_tupla[-1])  # Imprime: 40

mi_tupla2 = (1, 2, 3)
a, b, c = mi_tupla2
print(a, b, c)  # Imprime: 1 2 3

es_tupla = (5)  # ¿Esto es una tupla?

🟧 Tuplas - Métodos comunes

📌 Recordatorio:

Métodos comunes de las tuplas:

  • count(): Cuenta cuántas veces aparece un elemento.
  • index(): Devuelve el índice de la primera ocurrencia de un elemento.

En todos los iterables (listas, tuplas, cadenas, diccionarios), podemos verificar la existencia de un elemento dentro del iterable.

mi_tupla3 = (1, 2, 3, 1, 2, 1)
print(mi_tupla3.count(1))  # Imprime: 3
print(mi_tupla3.index(2))  # Imprime: 1

mi_tupla4 = ("rojo", "azul", "verde")
print("rojo" in mi_tupla4)  # Imprime: True

🟫 Cadenas (str)

Tipo inmutable que almacena secuencias de caracteres. Se usan comillas simples o dobles.

Funciones y operaciones comunes:

  • Concatenación.
  • Repetición.
  • Longitud.
  • Indexación y slicing.
texto = "Hola, Python!"
print(texto[0])      # Salida?
print(len(texto))    # Salida?
print(texto[7:])     # Salida?
print(texto * 3)     # Salida?

🔼 Diccionarios (dict)

Son estructuras de datos que almacenan pares de clave-valor.

Las claves deben ser únicas e inmutables (pueden ser cadenas, números o tuplas). Se crean con llaves {}.

Propiedades:

  • No ordenados (antes de Python 3.7). Desde Python 3.7, mantienen el orden de inserción.
  • Acceso rápido a los valores usando las claves.
  • Mutables.
alumno = {
    "nombre": "Ana",
    "edad": 21,
    "carrera": "Ingeniería en IA"
}

print(alumno["nombre"])  # Salida: "Ana"
alumno["promedio"] = 9.5  # Agregar nueva clave
alumno["edad"] = 22       # Modificar valor
print(alumno)

🔼 Diccionarios - Métodos comunes

📌 Recordatorio:

Métodos comunes de los diccionarios:

  • keys(): Devuelve las claves.
  • values(): Devuelve los valores.
  • items(): Devuelve los pares clave-valor como tuplas.
  • get(): Accede a un valor de forma segura (sin causar error si la clave no existe).
  • del: Elimina una clave y su valor.
  • pop(): Elimina una clave y devuelve su valor.

También podemos iterar sobre un diccionario:

  • Claves: for clave in diccionario.
  • Valores: for valor in diccionario.values().
  • Pares clave-valor: for clave, valor in diccionario.items().
del alumno["carrera"]
promedio = alumno.pop("promedio")
print(alumno)  # Salida: {"nombre": "Ana", "edad": 22}
print(alumno.keys())      # Salida: dict_keys(["nombre", "edad"])
print(alumno.get("edad")) # Salida: 22

for clave, valor in alumno.items():
    print(f"{clave}: {valor}") # Salida?

Operadores

Operador

Es un símbolo especial que realiza una operación en uno o más operandos.

Tipos de Operadores

Tipo Operadores Ejemplo
Aritméticos +, -, *, /, //, %, ** a + b
Relacionales ==, !=, >, <, >=, <= x >= 10
Lógicos and, or, not x > 5 and x < 20
Asignación =, +=, -=, *=, /= contador += 1
Membresía in, not in "a" in "cadena"
Identidad is, is not x is None

Reglas de precedencia

TIPS:

  • Usa paréntesis para agrupar expresiones y evitar errores de precedencia.
  • Potencia (**) se evalúa de derecha a izquierda.
  • Los operadores lógicos siguen el orden: not > and > or.
  • La asignación siempre ocurre al final.
Nivel Operador(es) Descripción Ejemplo
1 (más alta) () Agrupación mediante paréntesis (5 + 3) * 2 → 16
2 ** Potencia 2 ** 3 ** 2 → 512
3 +x, -x Operadores unarios -3 + 4 → 1
4 *, /, //, % Multiplicación, división, división entera, módulo 10 % 3 → 1
5 +, - Suma y resta 10 - 4 + 2 → 8
6 ==, !=, >, <, >=, <=, is, in Comparación, identidad y pertenencia a in lista
7 not Negación lógica not True → False
8 and Conjunción lógica True and False → False
9 or Disyunción lógica False or True → True
10 (más baja) =, +=, -=, *=, /= Asignación x = 5; x += 1 → 6

Aritméticos: Realizan operaciones matemáticas.

  • + (suma), - (resta), * (multiplicación), / (división), // (división entera), % (módulo o residuo), ** (exponenciación).
a = 10
b = 3
suma = a + b
resta = a - b
multiplicacion = a * b
division = a / b
division_entera = a // b
modulo = a % b
exponenciacion = a ** b

De Asignación: Asignan valores a variables.

  • = (asignación simple), += (suma y asignación), -= (resta y asignación), *= (multiplicación y asignación), /= (división y asignación), //= (división entera y asignación), %=(módulo y asignación), **=(exponenciación y asignación).
x = 5
x += 3  # Equivalente a x = x + 3

Relacionales: Comparan valores y devuelven un resultado booleano.

  • == (igual a), != (no igual a), < (menor que), > (mayor que), <= (menor o igual a), >= (mayor o igual a).
a = 10
b = 3
resultado = (a > b)  # True, ya que 10 es mayor que 3

Lógicos: Realizan operaciones lógicas en valores booleanos.

  • and (y lógico), or (o lógico), not (no lógico).
a = 10
b = 3
condicion1 = (a > b) and (b != 0)  # True, ambas condiciones son verdaderas
condicion2 = (a < b) or (b == 0)   # False, ninguna condición es verdadera

De Membresía: Verifican si un valor está presente en una secuencia.

  • in (está en), not in (no está en).
lista = [1, 2, 3, 4, 5]
pertenece = 3 in lista  # True, ya que 3 está en la lista
no_pertenece = 6 not in lista  # True, ya que 6 no está en la lista

De Identidad: Verifican si dos variables se refieren al mismo objeto en la memoria.

  • is (es), is not (no es).
a = [1, 2, 3]
b = a  # b ahora apunta al mismo objeto que a
c = [1, 2, 3]  # un nuevo objeto con el mismo contenido que a

print(a is b)  # True, ya que b apunta al mismo objeto que a
print(a is c)  # False, ya que c es un objeto diferente
print(a is not c)  # True, ya que a y c no apuntan al mismo objeto

Estructuras Condicionales

Permiten ejecutar bloques de código dependiendo si una condición es verdadera o falsa.

Estructura if: Ejecuta el bloque de código dentro del if si la condición es verdadera.

edad = 20
if edad >= 18:
    print("Eres mayor de edad.")

Estructura elif: Se utiliza para evaluar condiciones adicionales si la condición del if no se cumple.

temperatura = 25
if temperatura > 30:
    print("Hace mucho calor.")
elif temperatura > 20:
    print("La temperatura es agradable.")

Estructura else: Proporciona un bloque de código que se ejecuta si ninguna de las condiciones anteriores es verdadera.

edad = 15
if edad >= 18:
    print("Eres mayor de edad.")
else:
    print("Eres menor de edad.")

Estructuras de Iteración/Ciclos (for, while):

Permiten repetir bloques de código mientras se cumpla una condición.

Ciclo for:

Obtiene los elementos de un iterable (como una lista) y ejecuta el bloque de código en cada iteración.

frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
    print(fruta)

Ciclo while:

Ejecuta el bloque de código mientras la condición sea verdadera.

contador = 0
while contador < 5:
    print(contador)
    contador += 1

Estructuras de control

Tipo Estructura Descripción Ejemplo de uso
Condicional if Evalúa una condición y ejecuta un bloque si se cumple. if edad >= 18:
Condicional if ... else Ejecuta un bloque si se cumple la condición, otro si no. if edad >= 18: ... else: ...
Condicional if ... elif ... else Permite múltiples condiciones encadenadas. if x > 0: ... elif x < 0: ...
Ciclo while Repite un bloque mientras una condición sea verdadera. while i < 10:
Ciclo for Itera sobre elementos de una secuencia. for letra in palabra:
Control de flujo break Termina un ciclo de forma anticipada. if x == 5: break
Control de flujo continue Omite el resto del ciclo actual. if x == 0: continue
Control nulo pass No hace nada; marcador de posición. if condicion: pass

Tips: Usa if...elif...else para condiciones excluyentes. Prefiere for para recorrer elementos y while cuando no sabes cuántas iteraciones. Evita ciclos infinitos.

Funciones

Función

  • Una función es un bloque de código reutilizable que realiza una tarea específica.
  • Permite dividir un programa en partes más pequeñas, legibles y mantenibles.

Una función encapsula una acción o un conjunto de instrucciones que pueden ejecutarse varias veces sin reescribir el código.

¿Por qué usar funciones?

  • Evitan la repetición de código (DRY: Don’t Repeat Yourself).
  • Mejoran la organización del programa.
  • Permiten dividir problemas complejos en partes más manejables.
  • Facilitan las pruebas y el mantenimiento.

Sintaxis de funciones

def nombre_funcion(parámetros):
    """Docstring opcional que explica qué hace la función"""
    # cuerpo de la función
    return valor

Ejemplo:

def saludar(nombre):
    print(f"Hola, {nombre}!")

# llamada
saludar("Ana")

Funciones con retorno (return)

def cuadrado(x):
    return x * x

resultado = cuadrado(5)
print(resultado)  # Imprime 25

Buenas prácticas al definir funciones

  • Usa nombres descriptivos en minúsculas y con guiones bajos (snake_case).
  • Incluye una docstring que explique qué hace la función.
  • Limita la función a una sola responsabilidad.
  • Evita efectos secundarios innecesarios (modificar variables externas).
  • Usa return para valores que necesites reutilizar.
Tipo Descripción Ejemplo
Sin parámetros, sin retorno Ejecuta acciones, pero no recibe ni devuelve valores def bienvenida():
Con parámetros Recibe datos para trabajar con ellos def saludar(nombre):
Con retorno Devuelve un valor con return def suma(a, b): return a + b
Parámetros por defecto Asignan valores si no se proporcionan argumentos def saludar(nombre="invitado"):

¿Qué es un parámetro? ¿Qué es un argumento?

  • Parámetro: variable declarada en la definición de la función.

  • Argumento: valor real que se pasa cuando se llama a la función.

  • Ejemplo:

    def saludar(nombre):  # nombre es un parámetro
    print("Hola", nombre)

saludar("Lucía")  # "Lucía" es un argumento
